Nexus Protocol API

Immutable proof of existence on Polygon. No API key required.

Smart Contract

0xAFaC5727957C890BbFB99BC6b1A1961FDbe08040

witness(bytes32 hash) payable

Record a hash permanently on Polygon. Send 0.0003 POL.

Example: witness(0xabc123...)

verify(bytes32 hash, uint256 id) → bool

Check if a hash exists at a specific ID.

totalRecords() → uint256

Get total number of hashes recorded.

creatorRewards() → uint256

Accumulated creator rewards in wei.

Python SDK

pip install nexus-witness

from nexus import witness; witness("my data")

RPC Endpoint

Any Polygon RPC works. No permission needed.
